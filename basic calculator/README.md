# ✨ Online Calculator Hub

A modern, elegant, and user-friendly **Calculator Hub Web Application** that provides multiple calculators and unit converters in one place.
Designed with a clean UI, smooth animations, and interactive features to enhance user experience.

---

## 📌 Features

### 🧮 Core Calculator

* Basic arithmetic operations (+, −, ×, ÷)
* Real-time input display
* Error handling (e.g., divide by zero)
* Neon glow typing effect ✨

### 🌗 UI & Experience

* Modern **Glassmorphism Design**
* Gradient background with smooth animations
* **Dark / Light Mode Toggle**
* Responsive layout (works on all screen sizes)

### 🔊 Interactive Elements

* Button click sound effects
* Hover animations
* Smooth transitions

### 📜 History Feature

* Stores previous calculations
* Displays results dynamically

---

## 🔢 Additional Calculators

The hub includes multiple utility calculators:

* 📊 BMI Calculator
* 📈 Percentage Calculator
* 🌡️ Temperature Converter (Celsius → Fahrenheit)
* 📏 Length Converter (Meters → Kilometers)
* ⚖️ Weight Converter (KG → Pounds)
* ⏱️ Time Converter (Minutes → Hours)
